Installing Viking Minecraft

In order to install Viking Minecraft, follow these incredibly simple instructions.

  1. Download and save one of the following files (download links are just above here) to your desktop, or somewhere else easy to remember:
    • If you DO NOT have the mods already installed download the file called "Viking Minecraft v1.5.4 - Client Full Pack."
    • If you DO have the mods already installed download the file called "Viking Minecraft v1.5.4 - Client Update"
  2. Open up your Valheim folder by right-clicking on the game in Steam, hovering over "Manage" and left-clicking "Browse Local Files."
  3. Open the file you downloaded and drag all of the files/folders into the Valheim folder you opened above, choosing "Yes" or "OK" if it asks you to overwrite any files.
  4. Launch the game and BE A VIKING!
